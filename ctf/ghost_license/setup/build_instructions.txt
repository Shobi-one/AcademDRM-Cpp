Ghost License build instructions

From ctf/ghost_license/source:
1) cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
2) cmake --build build --config Release

- build/Release/GhostLicense.exe (MSVC multi-config)
- build/GhostLicense (single-config generators)
